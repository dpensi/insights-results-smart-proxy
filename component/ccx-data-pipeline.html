Insights Results Smart Proxy

[Front page] [Overall architecture]

Component: CCX Data Pipeline

Type

  • Service (filter)

Description

CCX Data Pipeline service intends to get Insights gathered archives and analyzes them using the Insights framework (OCP rules engine) in order to generate a report with the rules hit by the content of the archive.

This report is published back in order to be consumed by other services.

Interfaces

  • Input:
    • Archives send by Insights Operator
  • Output:
    • rule report in JSON format
