Component: Notification rules

Type

  • Data source

Description

Configuration parameters for CCX Notification Service in form of conditions used to determine which messages will be sent to Service Log or to Notification Backend. Additionally it is based on set of threshold values. Currently it is part of CCX Notification Service config files + config environment variables.
